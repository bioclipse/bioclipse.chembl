Help for ChEMBL wizard

( If the wizard is not open yet do File->New and choose chEMBL.)

1. Chose search perspective 'Compound' or 'Target' (1).

2. Chose what you want to search with (2).

3. Add text (3).

4. Push Search button (4).

5. Results are shown in the upper table (5)

6. To select an item to save add it to the lower table (6). This is done by drag and drop or check the boxes and push Select button(7).

7. Perform a new search, start at 1 again.

8. To delete item(s) from the lower table (6) simply check their boxes and push the Delete button (8).

9. When done click Next to chose where to save file and to name it. (4)
