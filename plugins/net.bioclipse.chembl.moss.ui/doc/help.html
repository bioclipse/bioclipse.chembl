Help for ChEMBL-MoSS wizard

( If the wizard is not open yet open it by File ->New choose CHEMBL-MoSS.)

1. First select a protein family from the top combo-box (1).

2. If you are online the available activities will be displayed in the box beneath. (2)

3. Choose one.

4. Now the table beneath should contain compounds from chosen family with a specified activity. (4)

If the total hit is under 50 compounds move to step 7.

5. The search is limited to a set of 50 compounds. If the set contain more than 50 compounds and you would like to work with all: simply press the button "Display all"(6).

6. You are also able to modify your limit in the limit box (3). If you use the arrows the limit will increase/decrease with 1 but you could also simply type in the number your self.

7. Simply click around in the boxes until you find what you are looking for.

8. Modify data by checking the Modify data checkbox (7).

9. To look at the activity intervall press the Graph button (8).

10. Update table by add lower (9) and upper (10) interval limits and push Update table (11).

11. When finish press Next

12. If a secondary dataset is desired start from 1. Other press Next again.

13. Chose where to put your file and name it.

Three files are saved under that name.
